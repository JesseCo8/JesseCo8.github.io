Radar Antenna Laser Calibration System

Overview

A production calibration system designed to fine-tune the resonant frequency of radar antennas used in motion-detector security systems. It ensures reliable performance across high-volume manufacturing.

Technologies Used

  • C# (automation and control)
  • Altium Designer (circuit design)
  • Pneumatics and laser integration

Features

  • Automated calibration of radar antenna resonance
  • Integrated safety interlocks for electrical, pneumatic, and laser systems
  • Production-ready documentation and certification

Explosive Cartridge Sensitivity Tester

Overview

A multi-channel test system capable of delivering controlled voltage, current, or power to 12 devices under test simultaneously, with integrated data logging for sensitivity and failure analysis.

Technologies Used

  • Custom power controller PCB (Altium Designer)
  • C# (power control software & UI)
  • Data acquisition systems

Features

  • Simultaneous testing of 12 devices under precise electrical conditions
  • Automated failure analysis data logging
  • Custom PCB and software integration

Final 31kWh Battery Tester

Overview

A standalone tester that communicates with the battery management system (BMS) to exercise all hardware functions, including charge/discharge cycles and monitoring of cell voltages and temperatures.

Technologies Used

  • C# (tester control & UI)
  • Embedded communication with BMS
  • Custom hardware interfaces for charge/discharge

Features

  • Automated charge and discharge testing
  • Reads and validates cell voltages and temperatures
  • Standalone deployment in production environments
