EV Battery Tester

Overview

A standalone tester that communicates with the battery management system (BMS) to exercise all hardware functions, including charge/discharge along with monitoring cell voltages and temperatures.

Technologies Used

  • Test automation in Python using PyTest
  • Labjack for digital control and analog measurements
  • Raspberry Pi Test Controller
  • Embedded CAN communication with DUT
  • 5-channel 2.4 kW discharge load with voltage sensing capability

Features

  • Automated charge and discharge testing
  • Reads and validates cell voltages and temperatures along with pack current
  • Validates battery heaters and auxillary voltage output
  • Touchscreen Display
  • Standalone deployment in production environments

Radar Antenna Laser Calibration System

Overview

A production calibration system designed to fine-tune the resonant frequency of radar antennas used in motion-detector security systems. It ensures reliable performance across high-volume manufacturing.

Jesse's Responsibilities

  • Complete Bill of Materials
  • Functional and Performance Certification documentation
